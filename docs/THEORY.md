# トリンクシーのふたり診断 診断理論の説明

## 1. 基本理論

### 3軸による性格測定モデル

トリンクシーのふたり診断は、**3つの独立した軸**で性格を測定する理論に基づいています。

#### 軸1: コミュニケーション軸（Communication Axis）
- **積極型 ↔ 受容型**
- 人との関わり方や会話のスタイルを測定
- **積極型**: 自分から話しかけ、話題を提供するタイプ
- **受容型**: 聞き役に回り、相手の話を受け止めるタイプ
- **バランス型**: 状況に応じて両方を使い分けるタイプ

#### 軸2: 意思決定軸（Decision Axis）
- **論理型 ↔ 感情型**
- 物事の判断や決断の仕方を測定
- **論理型**: データや事実を重視して判断するタイプ
- **感情型**: 直感や気持ちを重視して判断するタイプ
- **ハイブリッド型**: 状況に応じて両方を使い分けるタイプ

#### 軸3: 関係構築軸（Relationship Axis）
- **リード型 ↔ 寄り添い型**
- 人間関係での役割や立ち位置を測定
- **リード型**: 主導権を握り、方向性を示すタイプ
- **寄り添い型**: サポート役に回り、相手に寄り添うタイプ
- **対等型**: 対等な関係を築くタイプ

### 27タイプの分類

**3軸 × 3段階 = 27タイプ**

各軸が3段階（極端な2つ + 中間の1つ）なので、3 × 3 × 3 = 27通りの組み合わせが可能です。

例：
- `積極型_論理型_リード型`: 積極的に話し、論理的に判断し、リードするタイプ
- `受容型_感情型_寄り添い型`: 聞き役に回り、感情で判断し、寄り添うタイプ
- `バランス型_ハイブリッド型_対等型`: バランスを取り、柔軟に判断し、対等な関係を築くタイプ

## 2. スコア計算の理論

### 質問への回答からスコアを計算

各質問には5つの選択肢があり、それぞれにスコアが割り当てられています：
- **強く当てはまる**: +2点
- **やや当てはまる**: +1点
- **どちらでもない**: 0点
- **あまり当てはまらない**: -1点
- **全く当てはまらない**: -2点

### 各軸のスコア計算

**18問診断の場合:**
- 各軸6問ずつ（合計18問）
- 各軸のスコア範囲: **-12 ～ +12**
- 閾値: **±2**

**54問診断の場合:**
- 各軸18問ずつ（合計54問）
- 各軸のスコア範囲: **-36 ～ +36**
- 閾値: **±3**

### タイプ判定の理論

各軸のスコアから、以下のルールでタイプを判定します：

#### コミュニケーション軸
- **スコア >= 閾値**: 積極型
- **スコア <= -閾値**: 受容型
- **それ以外**: バランス型

#### 意思決定軸
- **スコア >= 閾値**: 論理型
- **スコア <= -閾値**: 感情型
- **それ以外**: ハイブリッド型

#### 関係性軸
- **スコア >= 閾値**: リード型
- **スコア <= -閾値**: 寄り添い型
- **それ以外**: 対等型

## 3. 相性判定の理論

### 各軸の相性概念

相性判定では、**各軸で異なる相性概念**を適用します：

#### 軸1（コミュニケーション）: **補完性重視**
- **積極型 × 受容型**: 100点（最高）
  - 理由: 一方が話題を提供し、もう一方が深く聞くことで、会話が自然に流れる
- **積極型 × バランス型 / 受容型 × バランス型**: 70点
- **積極型 × 積極型 / 受容型 × 受容型**: 50点
- **バランス型 × バランス型**: 80点

**理論的根拠**: コミュニケーションでは、話す役と聞く役が補完し合うことで、より深い理解が生まれます。

#### 軸2（意思決定）: **類似性重視**
- **同じタイプ同士**: 100点（最高）
  - 理由: 判断基準が似ているため、意思決定がスムーズで衝突が少ない
- **ハイブリッド型を含む組み合わせ**: 80点
- **論理型 × 感情型**: 40点（最も低い）
  - 理由: 判断基準が正反対のため、意見が対立しやすい

**理論的根拠**: 意思決定では、同じ価値観や判断基準を持つことで、スムーズに物事を進められます。

#### 軸3（関係性）: **補完性重視**
- **リード型 × 寄り添い型**: 100点（最高）
  - 理由: 一方が方向性を示し、もう一方がサポートすることで、役割分担が明確
- **リード型 × 対等型 / 寄り添い型 × 対等型**: 70点
- **リード型 × リード型 / 寄り添い型 × 寄り添い型**: 50点
- **対等型 × 対等型**: 80点

**理論的根拠**: 関係性では、リード役とサポート役が補完し合うことで、バランスの取れた関係を築けます。

### 総合相性スコアの計算式

```
総合スコア = 0.3 × 軸1の相性 + 0.4 × 軸2の相性 + 0.3 × 軸3の相性
```

**重み付けの理由:**
- **軸2（意思決定）が40%**: 価値観や判断基準の一致は、長期的な関係において最も重要
- **軸1（コミュニケーション）が30%**: 会話のスタイルは日常的な関係に影響
- **軸3（関係性）が30%**: 役割分担は関係のバランスに影響

### スコアの正規化

計算された総合スコア（46～100点）を、1～100点の範囲に正規化します：

```
正規化スコア = ((生スコア - 46) / 54) × 99 + 1
```

これにより、すべての組み合わせが1～100点の範囲で評価されます。

## 4. 科学的根拠

### MBTIとの関係

この診断は、**MBTI（Myers-Briggs Type Indicator）**の理論を参考にしていますが、独自の3軸モデルを採用しています。

**MBTIとの違い:**
- MBTI: 4軸（E/I, S/N, T/F, J/P）→ 16タイプ
- トリンクシーのふたり診断: 3軸（Communication, Decision, Relationship）→ 27タイプ

**トリンクシーのふたり診断の特徴:**
- よりシンプルで理解しやすい
- 恋愛・人間関係に特化した軸設定
- 各軸で異なる相性概念を適用（補完性 vs 類似性）

### 心理学的研究に基づく理論

#### 補完性理論（Complementarity Theory）
- **コミュニケーション軸と関係性軸**: 異なる特性が補完し合うことで、より良い関係を築ける
- 研究: 会話スタイルや役割分担の違いが、関係の満足度に影響することが示されている

#### 類似性理論（Similarity Theory）
- **意思決定軸**: 価値観や判断基準が似ていることで、衝突が少なく、理解しやすい
- 研究: 価値観の一致は、長期的な関係の満足度に最も影響することが示されている

### 質問設計の理論

各軸に対応する質問は、以下の原則に基づいて設計されています：

1. **明確な軸の測定**: 各質問は特定の軸のみを測定する
2. **均等な配分**: 各軸に同じ数の質問を割り当てる
3. **多角的な測定**: 様々な状況や場面での行動を質問する
4. **スコアの一貫性**: 同じ特性を示す回答には同じ方向のスコアを付与

## 5. 診断の正確性を保証する仕組み

### 質問データの構造

各質問には`axis`フィールドがあり、どの軸を測定するかが明確に定義されています：

```json
{
  "id": 1,
  "axis": "communication",
  "text": "初対面の人と2人きりになったとき、あなたは自分から積極的に話しかけますか？",
  "options": [
    { "label": "強く当てはまる", "score": 2 },
    { "label": "やや当てはまる", "score": 1 },
    { "label": "どちらでもない", "score": 0 },
    { "label": "あまり当てはまらない", "score": -1 },
    { "label": "全く当てはまらない", "score": -2 }
  ]
}
```

### 計算ロジックの正確性

- **質問データの`axis`フィールドを使用**: 質問の順序が変わっても正しく動作
- **理論に基づいた閾値設定**: 各軸のスコア範囲に応じて適切な閾値を設定
- **全27タイプの生成可能性**: どの回答パターンでも全27タイプが生成可能

### 均等性の確保

- **閾値の調整**: 実際の回答パターンを考慮して、より均等な分布になるように閾値を調整
- **質問数の配分**: 各軸に同じ数の質問を割り当てることで、各軸の測定精度を均等に

## 6. まとめ

トリンクシーのふたり診断の診断理論は、以下の要素で構成されています：

1. **3軸による性格測定**: コミュニケーション、意思決定、関係性の3つの軸で性格を測定
2. **27タイプの分類**: 3軸 × 3段階 = 27通りの組み合わせ
3. **科学的な相性判定**: 各軸で異なる相性概念（補完性 vs 類似性）を適用
4. **正確なスコア計算**: 質問データの`axis`フィールドを使用した科学的な計算
5. **理論に基づいた閾値設定**: 実際の回答パターンを考慮した適切な閾値

この理論により、**MBTIレベルの精度**で相性を診断できるようになっています。
